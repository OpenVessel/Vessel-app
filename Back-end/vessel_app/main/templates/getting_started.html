Getting Started

OpenVessel platform allows you to upload DICOM files from CT scans to the browser and utilize the power of machine learning to process your scans in various ways. In the future, we will offer support for various modalities such as MRI and X-ray. With OpenVessel you can process your data in mere minutes with just a few clicks.

Upload Page

After you login, you can navigate to the upload page. Enter the name of the study and a description on the left hand side.

Do not put in identifiable information.

Drag and drop the CT scans or DICOM files into the drag and drop box. Then left click the Submit to finalize the upload.

File Browser Page

The browser page will display every study uploaded to the database, only associated to the user who uploaded the data.

A thumbnail is generated from the files uploaded. Additionally, the file count for each upload is made visible.

The uploaded files are anonymized. However, you can view select metadata in the browser, including Study ID, Patient ID, and Study Date.


Browser Buttons

  1. Submit to Machine Learning - Here you can submit data into the machine learning pipeline to generate a 3d-rendered result.
  2. Render past 3D model - Past results of machine learning can be viewed at ay time via a dropdown box.
  3. Delete Upload - Delete the entire study and all associated generated models.

Submit a study to a machine learning model

From here you can submit an upload to be evaluated by the machine learning. Different parameters exist for different models, which allow you to change how the model performs.

3D render

The 3D render page uses VTK.js to render .vti files (volumetric files).
